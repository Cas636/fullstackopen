# Bloglist Frontend (Parte 7)

Frontend de la aplicacion Bloglist con React, Vite, React Router y React Query.

## Instalacion

```bash
npm install
```

## Ejecucion

```bash
npm run dev
```

El servidor de desarrollo iniciara en `http://localhost:5173`

## Tests

```bash
npm test           # Tests unitarios
npm run lint      # ESLint
```

## Dependencies

- react
- react-dom
- react-bootstrap
- bootstrap
- axios
- @tanstack/react-query
- react-router-dom

## Notas

- Requiere que el backend este corriendo en `http://localhost:3003`
- Sin autologin (requiere login manual)
- React Query para gestion de estado
- React Router para navegacion
- Bootstrap para estilos UI
- Vista de blogs, usuarios y detalles
